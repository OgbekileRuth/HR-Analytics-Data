# HR Analytics Project

## Overview
This project analyzes HR data using SQL to gain insights into employee performance, retention, and salaries.

## Tools Used
- SQL Server
- Excel
- Power BI

## Project Files
- HR Data.csv
- SQL Queries.sql
- Power BI Dashboard.pbix

## Key Analyses
- Employee count by department
- Salary analysis
- Employee retention
- Performance analysis
